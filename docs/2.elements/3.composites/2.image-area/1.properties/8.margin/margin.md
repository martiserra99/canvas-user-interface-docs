nav: margin
title: margin
---

It is the margin of the image in its area. It is an object with these properties:

- **top**: Top margin.
- **right**: Right margin.
- **bottom**: Bottom margin.
- **left**: Left margin.

Its default value is:

```javascript
{
  top: 0,
  right: 0,
  bottom: 0,
  left: 0
}
```
